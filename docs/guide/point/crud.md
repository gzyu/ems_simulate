# 测点增删改查 (Point CRUD)

本章节介绍如何在 EMS Simulate 中管理测点（Points），包括添加、查看、编辑和删除操作。

## 1. 添加测点 (Create)

在设备详情页面，选择目标从机（Slave），点击“添加测点”按钮。

![](/doc-img/point/crud/1.png)

在弹出的对话框中填写测点信息：
- **名称**：测点的描述性名称。
- **地址**：寄存器起始地址。
- **寄存器类型**：如 Holding Register (保持寄存器) 或 Input Register (输入寄存器)。
- **数据类型**：如 Int16, Float32 等。
- **解析公式**：可选，用于对原始值进行线性变换。

![](/doc-img/point/crud/2.png)

## 2. 查看测点 (Read)

添加成功后，测点将显示在数据表格中。您可以查看其实时值、原始值及其他属性。

## 3. 编辑测点 (Update)

拉开表格抽屉，选择属性编辑，可以修改测点的各项属性。

![](/doc-img/point/crud/3.png)

注意：修改地址或数据类型可能会影响数据的正确解析。

## 4. 删除测点 (Delete)

点击测点行右侧的“删除”按钮即可移除该测点。也可以使用“清空测点”功能批量删除当前从机下的所有测点。

![](/doc-img/point/crud/4.png)
