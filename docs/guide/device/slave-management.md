# 从机管理

在此页面可以对设备的从机进行管理，包括添加、编辑和删除从机配置（只针对Modbus协议和IEC104协议）。

## 1. 添加从机

点击界面中的添加从机按钮，在弹出的表单中填写从机地址。填写完毕后点击“确定”即可完成从机的添加。

>![](/doc-img/device/slave-management/1.png)

## 2. 编辑从机

在从机列表中，找到需要修改的从机项。点击从机旁边的三个点，此时会弹出从机信息编辑选项，点击编辑从机。

>![](/doc-img/device/slave-management/2.png)

修改从机地址后，点击“确定”以更新从机数据。

>![](/doc-img/device/slave-management/3.png)

## 3. 删除从机

在从机列表中，找到需要修改的从机项。点击从机旁边的三个点，此时会弹出删除从机选项，点击删除从机。

>![](/doc-img/device/slave-management/4.png)
