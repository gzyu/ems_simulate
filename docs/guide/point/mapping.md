# 测点映射 (Point Mapping)

测点映射功能允许将一个测点的值直接映射到另一个测点，实现数据的联动或镜像。

## 1. 应用场景

- **设备联动**：当设备 A 的状态改变时，自动触发设备 B 的状态改变（可支持多个设备）。

  - 场景1：A测点的值由B和C测点的值决定

    举例：
    $$
    PCS视在功率 = \sqrt{PCS有功功率^2 + PCS无功功率^2}
    $$

  - 场景2：A测点的值受B测点的值影响

    举例：当PCS开关机时，开关机测点的值会影响运行状态测点的值。

  - 场景3：设备具有不同协议，需要跨设备数据真实场景模拟

    储能计量电表（DL/T645）的实时功率约等于所有PCS（Modbus）的有功功率之和。

- **数据镜像**：将重要数据同时映射到多个寄存器地址供不同系统读取。

## 2. 创建映射

进入“测点映射”管理页面，点击“创建映射”。

![](/doc-img/point/mapping/1.png)

配置源测点和目标测点：
- **源测点** (Source)：数据的提供方。
- **目标测点** (Target)：数据的接收方，即当前测点。

通过设备树选择测点

![](/doc-img/point/mapping/2.png)

选择完测点后，输入别名（不能忽略，公式通过别名进行运算）

![](/doc-img/point/mapping/3.png)

输入完公式后进行保存，以上图为例，电表的正向有功总电能=p1+p2

## 3. 映射规则

- 映射是**单向**的：源测点的值变化会更新目标测点，反之则不会。
- 映射是**实时**的：系统检测到源测点值变更后，会立即同步到目标测点。
