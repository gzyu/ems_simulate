# 测点模拟 (Point Simulation)

EMS Simulate 支持对测点值进行动态模拟，方便测试上位机的监控报警功能。

## 1. 模拟原理

开启模拟后，系统会根据预设的规则自动更新测点的值。支持的模拟类型包括：
- **随机数** (Random)：在指定范围内生成随机值。
- **递增** (Increment)：按固定步长递增。
- **递减** (Decrement)：按固定步长递减。
- **正弦波** (Sine)：按正弦曲线变化。

## 2. 配置模拟

打开测点的下拉抽屉，选择仿真模拟，进入模拟配置界面。

![](/doc-img/point/simulation/1.png)

选择模拟类型并设置参数（此设置只针对单个测点）：
- **最小值/最大值**：限制数值范围。
- **步长**：每次变化的幅度。
- **周期**：正弦波的周期（秒）。

## 3. 启动与停止

配置完成后，点击“开始模拟”。此时测点值将按照规则自动变化。关闭模拟开关即可停止更新。

![](/doc-img/point/simulation/2.png)

注意事项：最上方的模拟针对于所有测点，当更改过某个测点的模拟方式之后，该测点按照测点本身模拟方式进行模拟。
